# kitso_line_bot

## 概要
Google Apps Script (GAS) と Google スプレッドシートを組み合わせて構築された LINE Bot アプリケーションです。
サークルや部活動（九工大交響楽団）での利用を想定しており、メンバーの予定確認、部室や講義室の鍵の管理、イベントまでのカウントダウン、リマインダー機能、さらに ChatGPT API を利用した雑談や音楽 Tips 機能を統合しています。

## 主な機能

- **スケジュール確認 (`schedule`)**
  - 連携した Google カレンダー（[EMAIL_ADDRESS]）から今日や今週の練習予定を取得し、合奏の有無や個人練習日・開放日などをお知らせします。
- **鍵の管理 (`WheresKey`)**
  - 「500人講義室」および「部室」の鍵の貸出・返却を LINE 上から記録できます。
  - 「いまどこ？」ボタンを利用することで、現在誰が持ち出しているかをすぐに確認できます。
- **イベントカウントダウン (`Just??toDaystoEvent`)**
  - 定期演奏会などのイベント本番までの残り日数をカウントダウンしてお知らせします。
- **リマインダー機能 (`Reminder`)**
  - 指定した日付でリマインダーを登録、確認、削除することができます。
  - 未完了のリマインダーを一覧形式で表示することも可能です。
- **音楽 Tips (`MusicTips`)**
  - ChatGPT を利用して、オーケストラにまつわる雑学や豆知識を自動生成して教えてくれます。
- **AI チャット応答**
  - 登録された専用コマンド以外のメッセージを送信すると、ChatGPT が自由に応答してくれます。

## システム構成

- **バックエンド / データベース**: Google Apps Script (GAS), Google スプレッドシート
  - 以下のシートをデータベースとして活用しています：
    - `userID` : ユーザー固有IDと名前の紐付け
    - `Log` : すべてのアクションやエラーのログ
    - `KeyLog` : 鍵の貸出・返却履歴
    - `reminder` : 登録されたリマインダー一覧
- **フロントエンド / UI**: LINE Messaging API
- **外部 API 連携**: 
  - Google Calendar API (スケジュール取得)
  - OpenAI API (ChatGPT 応答)

## 基本的な使い方

1. **ユーザー登録**: 
   初めて利用する際は、LINEのトーク画面に以下の形式で送信してユーザー名を登録します。
   ```
   userName
   （あなたの名前）
   ```
2. **メニューの操作**: 
   LINE リッチメニュー等のボタンから「HowtoUse（使い方）」「schedule（スケジュール）」「WheresKey（鍵の管理）」などの機能を利用します。
3. **リマインダーの登録**:
   ```
   reminder
   2023/11/04
   （リマインドしたい内容）
   ```
   のように改行で区切って送信することで登録が可能です。
